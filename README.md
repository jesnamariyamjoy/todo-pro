# 🚀 Pro Task Manager (Advanced To-Do App)

A modern, feature-rich task management web application built using **HTML, CSS, and JavaScript**, designed to improve productivity with powerful features like analytics, reminders, and drag-and-drop functionality.

---

## 🌟 Key Features

### 📝 Task Management
- Add, delete, and update tasks
- Mark tasks as completed or pending
- Reorder tasks using drag-and-drop

### 🔍 Smart Search & Filtering
- Real-time task search
- Filter tasks by:
  - All
  - Completed
  - Pending

### 📅 Due Date Tracking
- Assign due dates to tasks
- Visual indication of deadlines

### 🔔 Notifications & Reminders
- Browser notifications for due tasks
- Automatic reminder system

### 📊 Analytics Dashboard
- Real-time task statistics
- Visual representation using charts (Chart.js)
- Track:
  - Total tasks
  - Completed tasks
  - Pending tasks

### 🌙 Dark Mode
- Toggle between light and dark themes
- Improved user experience for night usage

### 💾 Data Persistence
- Stores tasks in browser using Local Storage
- No data loss on refresh

### 🎨 Modern UI/UX
- Glassmorphism design
- Responsive layout
- Smooth animations and transitions
